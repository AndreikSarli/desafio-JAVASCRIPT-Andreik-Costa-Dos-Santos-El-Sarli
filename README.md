# 🚗 Ford Ranger 2022 \| Desafio JavaScript

Projeto desenvolvido como desafio prático utilizando **HTML, CSS e
JavaScript puro**, com foco em:

-   📱 Responsividade\
-   ⚡ Interações dinâmicas\
-   🎨 Interface moderna\
-   🧠 Manipulação do DOM\
-   🛠️ Organização de código

------------------------------------------------------------------------

## 📖 Sobre o projeto

O projeto simula uma **landing page da Ford Ranger 2022**, contendo:

-   ✅ Página inicial responsiva\
-   ✅ Página de lançamento dos veículos\
-   ✅ Comparador dinâmico de carros\
-   ✅ Formulário de contato funcional\
-   ✅ Modal interativo\
-   ✅ Máscaras de CPF e telefone\
-   ✅ Publicação com GitHub Pages

------------------------------------------------------------------------

## 🚀 Funcionalidades

### 🏠 Home

-   Banner principal responsivo\
-   Navegação entre páginas\
-   Layout adaptável para mobile

------------------------------------------------------------------------

### 🚘 Lançamento

-   Exibição dos modelos Ford Ranger\
-   Seleção de veículos para comparação\
-   Modal de comparação dinâmica\
-   Overlay ao abrir comparação\
-   Fechamento por:
    -   Botão\
    -   Clique fora\
    -   Tecla **ESC**\
-   Tabela responsiva no mobile

------------------------------------------------------------------------

### 📩 Formulário de contato

-   ✅ Validação HTML5\
-   📞 Máscara dinâmica de telefone\
-   🪪 Máscara dinâmica de CPF\
-   ⚡ Captura de dados via JavaScript\
-   💬 Mensagem de sucesso\
-   🧱 Classe `Contato`\
-   🖥️ Exibição dos dados no console
